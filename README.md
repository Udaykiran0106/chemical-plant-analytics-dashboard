# Chemical Plant Operations Analytics Dashboard

## Project Overview
This project is a **chemical plant operations data analysis dashboard**
built using **SQL and Python**.  
It analyzes production, energy consumption, yield, downtime, and quality data
and presents insights in a **single, clear dashboard image**.

## Tools Used
- Python
- SQLite
- SQL
- pandas
- matplotlib

## Dashboard Insights
- Total Production
- Average Yield
- Average Energy Consumption
- Total Downtime
- Unit-wise Production
- Energy Consumption by Unit
- Downtime by Unit
- Quality Status (OK vs Off-spec)
- Monthly Production Trend
